# مقایسه الگوریتم‌های طبقه‌بندی روی مجموعه داده Iris

## معرفی پروژه
در این پروژه، عملکرد دو الگوریتم مشهور یادگیری ماشین بر روی دیتاست گل زنبق (Iris Dataset) بررسی و مقایسه شده است. هدف اصلی، تقسیم استاندارد داده‌ها به دو بخش **آموزش (Train)** و **تست (Test)** برای ارزیابی دقیق مدل‌هاست.

الگوریتم‌های استفاده شده:
1. **Gaussian Naive Bayes (GaussianNB)**
2. **K-Nearest Neighbors (KNN)**

## مراحل انجام کار
در فایل `iris_comparison.py` مراحل زیر پیاده‌سازی شده است:
1. بارگذاری داده‌های Iris از کتابخانه `sklearn`.
2. تقسیم داده‌ها به صورت **۷۰٪ آموزش** و **۳۰٪ تست** (با استفاده از `train_test_split`).
3. آموزش مدل‌ها با داده‌های آموزشی.
4. محاسبه و مقایسه دقت (Accuracy) مدل‌ها بر روی داده‌های تست.

## نتایج مقایسه
پس از اجرای کد (با `random_state=42`)، نتایج زیر حاصل می‌شود:

| نام مدل | دقت (Accuracy) | وضعیت |
| :--- | :---: | :--- |
| **GaussianNB** | ~۹۷.۷٪ | بسیار خوب |
| **KNN (k=3)** | ~۱۰۰٪ | عالی |

*نکته: در این تقسیم‌بندی خاص، الگوریتم KNN عملکرد کمی بهتری نسبت به بیزین گوسی نشان داد.*
